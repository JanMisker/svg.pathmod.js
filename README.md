# svg.pathmod.js

A plugin for the [svgdotjs.github.io](https://svgdotjs.github.io/) library to modify SVG paths.
The core of this plugin is based on [simplify-js](https://github.com/mourner/simplify-js) (c) 2017, Vladimir Agafonkin
